## Bright-SQL-Exercises Overview
This repository contains a collection of my original, handwritten class exercises and notes completed as part of my learning journey. These exercises reflect my understanding of key concepts, problem-solving approaches, and practical applications covered during class sessions. Each query starts with a question, SQL script, then followed by the output.

## Purpose
- Reinforce theoretical concepts through practice
- Improve problem-solving and analytical skills
- Maintain a structured record of learning progress
- Provide quick reference material for revision


## Key Learning Outcomes
- Better understanding of core concepts
- Improved accuracy in solving problems
- Ability to explain solutions step-by-step
- Strong foundation for advanced topics

## Summary
This repository showcases consistent effort in practicing and understanding concepts through handwritten exercises. It serves as a valuable resource for revision and demonstrates a hands-on approach to learning.
